* Description
Given an integer, write a function to determine if it is a power of three.

Example 1:

Input: 27
Output: true

Example 2:

Input: 0
Output: false

Example 3:

Input: 9
Output: true

Example 4:

Input: 45
Output: false

Follow up:
Could you do it without using any loop / recursion?

* Solution
#+begin_src cpp
  #include <iostream>
  #include <math.h>
  using namespace std;

  class Solution {
  public:
    bool isPowerOfThree(int n) {
      if (n == 0)
        return false;
      float e = log(n) / log(3);
      if (n == pow(3, e))
        return true;
      return false;
    }
  };
#+end_src
